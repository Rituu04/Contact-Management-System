# Contact Management System

## Description
This is a simple Contact Management System developed in C language using file handling.

## Features
- Add Contact
- Edit Contact
- Search Contact
- List Contact
- Delete Contact

## Technologies Used
- C Programming
- File Handling
- Structures
- Functions

## Compiler Used
- Turbo C / Dev-C++

## Note
The application stores contact details in a file named `contact.txt`.
